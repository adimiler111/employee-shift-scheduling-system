# Employee Shift Scheduling System

A Python and Streamlit-based system for automatic employee shift scheduling.

This system assigns employees to shifts based on availability and operational needs.

## Technologies
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- PostgreSQL

## Features
- Employee login system
- Excel file upload
- Automatic shift assignment
- Data visualization
- 
## How the System Works

1. The user logs into the system.
2. An Excel file containing employee data and preferences is uploaded.
3. The system processes the data using Python.
4. Employees are automatically assigned to shifts.
5. The final shift schedule is displayed in the dashboard.
